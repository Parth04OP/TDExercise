# TDExercise
Task1.py:
In this file, the program first asks the user to enter two numbers.These values are stored in variables and then used to perform different arithmetic operations.
It performs following operations:
1.Addition
2.Subtraction
3.Multiplication
4.Division
Each result is stored in a separate variable and then printed on the screen.

Task2.py:
In this file, the program asks the user to enter their first name and last name.Both inputs are stored as strings and then joined together to form a full name.After combining the names, the program prints a greeting message using the full name.This program mainly demonstrates how string input and concatenation work in Python.
